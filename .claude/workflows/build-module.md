# Workflow: Build New Module

## Skill chain
product-owner → business-analyst → solution-architect → service-layer-architect → rbac-expert → qa-architect → release-manager

## Prompt

```text
Use skills:
- product-owner
- business-analyst
- solution-architect
- service-layer-architect
- rbac-expert
- qa-architect

Design and break down module: [module name].
Return requirements, data model, permissions, services, UI routes, tasks, and tests.
```
